# <a name="using-the-groups-feature-to-create-private-events"></a>Uso de la característica Grupos para crear eventos privados

Nuestra característica grupos ofrece a los hosts la capacidad de crear un espacio de realidad virtual abierto a una lista aprobada de invitados. Los hosts pueden agregar manualmente los nombres de usuario o correos electrónicos de AltspaceVR a los que les gustaría conceder acceso a su espacio. Aquí se muestra cómo puede usar esta característica para hospedar eventos, reuniones y conferencias:

> [!NOTE]
> Para poder acceder a "Grupos", debe participar en las características de acceso anticipado iniciando sesión en altvr.com, editando el perfil de usuario en el sitio web y seleccionando Acceso anticipado. O bien, dentro de la aplicación AltspaceVR, Configuración, General, Participar en el programa de acceso anticipado.

1. En el explorador, vaya a  [https://altvr.com](https://altvr.com)   e inicie sesión en su cuenta. 
2. Busque el **menú** desplegable Más.
3. Seleccionar **grupos**

![AltspaceVR web view with more pull down menu highlighted (AltspaceVR web view with more pull down menu highlighted )AltspaceVR web view with more pull down menu highlighted](images/group-features-img-01.png)

4. Una vez en la página Grupos, puede ver una lista de grupos de los que forma parte o de los que es administrador. 

![Página de grupos de vistas web altspaceVR](images/group-features-img-02.png)

5. Seleccione el botón **Crear** para crear un nuevo grupo.
6. Rellene los campos para crear el grupo.

![Nueva página de creación de grupos con campos de entrada](images/group-features-img-03.png)

> [!NOTE]
> Puede mantener el puntero sobre el verde "¿?" para obtener información adicional sobre los campos que los tienen.

7. Envíe un correo electrónico a todos los asistentes y haga que creen una cuenta altspaceVR con la misma dirección de correo electrónico. Pueden crear una cuenta altspaceVR en el mundo o en nuestro sitio [web.](https://account.altvr.com/users/sign_up)

## <a name="how-to-create-a-private-eventfrom-a-group"></a>Cómo crear un evento privado a partir de un grupo 

1. Una vez en la página Miembro o Administrador, seleccione el grupo al que desea acceder. 
2. **Como administrador,** puede modificar todas las opciones que configuró anteriormente para crear el grupo.Vaya al botón "Editar" de la página del grupo para acceder a ellos.  
3. **Agregar otros administradores para un grupo:**   Al editar, puede agregar otros usuarios como administradores para ayudar a administrar el grupo.  
    * Debe ser el nombre de usuario altspaceVR del usuario y no el correo electrónico de su cuenta.  
4. Seleccione **Crear evento**.  

![Evento creado](images/group-features-img-04.png)

## <a name="how-to-attend-a-group-event-as-an-invitee"></a>Cómo participar en un evento de grupo, como invitado 

1. Abra el menú principal en la aplicación AltspaceVR: 

![Menú principal de la aplicación AltspaceVR](images/group-features-img-05.png)

2. Seleccione **Eventos** en la parte inferior del menú.
3. En la parte superior, busque **Mis eventos.**
4. Busque el evento por su **nombre** proporcionado por el creador de eventos en la vista principal con las imágenes de mosaico.Selecciónelo para escribir el evento si está en directo.

![Aplicación AltspaceVR con el panel mis eventos resaltado](images/group-features-img-06.png)

## <a name="how-to-find-the-group-world-commons"></a>Cómo encontrar el mundo del grupo (Commons):  

1. Abra el menú principal:  
2. Seleccione **Mundos** en la parte inferior del menú.   
3. En la parte superior, busque **Mis mundos.**
4. Busque el nombre del grupo y selecciónelo para escribir el mundo del grupo.

![Aplicación AltspaceVR con el panel Mis mundos resaltado](images/group-features-img-07.png)

## <a name="how-to-add-an-email-list-to-a-group"></a>Cómo agregar una lista de correo electrónico a un grupo 

Si tiene un grupo grande y necesita agregarlos de forma masiva, primero deberá crear una lista de direcciones de correo electrónico para todas las asociadas a su grupo. Tenga en cuenta que si se sucriben a La vinculación de cuentas Microsoft, necesitará su dirección de cuenta Microsoft para que este paso funcione.

### <a name="creating-a-list"></a>Creación de una lista

Use un programa de hoja de cálculo para crear una sola columna con todas las direcciones de correo electrónico. Una vez que lo haya hecho, haga lo siguiente:  

1. Cree o edite el grupo.  
2. Busque el **campo Correos electrónicos de** miembro.  
3. Busque la **opción Choose File (Elegir** archivo) para importar los correos electrónicos de un archivo *.csv.
4. La lista debe importarse con un correo electrónico por línea.

> [!NOTE]
> Si lo elige, también puede copiar y pegar correos electrónicos en el campo en lugar de importar desde un archivo *.csv.

### <a name="importing-a-list"></a>Importar una lista

Algunas plataformas pueden permitirle exportar un archivo *.csv (valor separado por comas) que tiene una columna de "correo electrónico" dedicada.Puede seleccionar esta columna para usarla.Una vez que esté disponible, los pasos son principalmente los mismos:  

1. Cree o edite el grupo.  
2. Busque el **campo Correos electrónicos de** miembro.  
3. Busque la **opción Choose File (Elegir** archivo) para importar los correos electrónicos de un archivo *.csv.
4. La lista debe importarse con un correo electrónico por línea.

> [!NOTE]
> Si lo elige, también puede copiar y pegar correos electrónicos en el campo en lugar de importar desde un archivo *.csv.

## <a name="faq"></a>Preguntas más frecuentes 

### <a name="im-having-trouble-filling-out-the-fields-in-groups-what-do-they-mean"></a>Tengo problemas para rellenar los campos en grupos. ¿Qué significan?

(*Puede mantener el puntero sobre el verde "?"para obtener información adicional sobre los campos que los tienen). 

* **Nombre**   : nombre del grupo que será visible para todos los miembros.   
* **Descripción**   : campo para describir el propósito del grupo. Visible para todos los miembros.
* **Usuarios**   : este campo permite enumerar los miembros del grupo como un nombre de usuario o una dirección de correo electrónico, siempre que sea uno por línea.Es una manera de definir los miembros del grupo.
    * Rellene un nombre de usuario o un correo electrónico por línea.  
    * Para buscar un nombre de usuario en AltspaceVR, haga que el usuario abra el "Menú principal" y vaya a la sección "Me".Es el nombre entre paréntesis.  
    * Para buscar el nombre de usuario en el sitio web, haga que el usuario vaya a "Perfil" en la opción "Más -> Perfil". Puede encontrarlo en la columna izquierda de la página Perfil.  
    * El campo distingue mayúsculas de minúsculas, por lo que "username" y "Username" se consideran diferentes.  
    * Este campo acepta nombres de usuario y correos electrónicos asociados a cuentas válidas.Asegúrese de que hay un elemento por línea.No es necesario colocar un correo electrónico o un nombre de usuario si es la misma persona, ya que tampoco funcionará.  
    * Las cuentas deben estar en buen estado.  
    * La lista se puede buscar con la opción de búsqueda del explorador "Ctrl + F".  
* **Imagen**   : se usa como imagen de banner promocional para el grupo.Está visible en la página web grupo y en VR.  
    * Se recomienda 1920 x1080 para mejorar la fidelidad de la imagen.  
    * Se prefiere JPEG o PNG.  
    * Las imágenes deben tener una relación de aspecto de 16:9.  
    * Generamos automáticamente miniaturas de la carga y las usamos en varias pantallas de interfaz de usuario en AltspaceVR y el sitio web.  
* **Imagen de perfil**   : se usa como imagen de perfil de grupos. Está visible en la página web grupo y en VR.  
    * Se recomienda 512×512 para mejorar la fidelidad de la imagen.  
    * Se prefiere JPEG o PNG.  
    * Generamos automáticamente miniaturas de la carga y las usamos en varias pantallas de interfaz de usuario en AltspaceVR y el sitio web.  
* **Seleccionar un entorno**   : en el lado derecho de la página, verá varias miniaturas de los entornos disponibles que se usarán. Este entorno actuará como el mundo principal del grupo y aparecerá en "Worlds -> My Worlds -> <Name from "Name" field> Commons" (Mundos -> Mis mundos -> <Nombre del campo "Nombre"> Commons) para usted y sus miembros del grupo.  
    * Si desea crear su propio mundo, debe hacerlo a través del sitio web [,](https://account.altvr.com/worlds/my)verá esos mundos como opciones posibles. 

> [!NOTE]
> Esta característica está en versión beta. Animamos a los grupos a que utilicen mundos preexistentes en el cuadro de diálogo Selección del entorno.

* La palabra "Commons" siempre aparecerá como sufijo en el mundo de los juegos de palabras de cualquier grupo.  
* **Mensajes de correo electrónico de miembro**   : este campo le permite enumerar los usuarios por el correo electrónico asociado a una cuenta altspaceVR.  **El uso de esta lista invalida la lista "Usuarios" descrita anteriormente.**   Esta es la manera recomendada de administrar la pertenencia a un grupo.  
    * Al rellenar este campo, asegúrese de que el correo electrónico asociado a la cuenta altspaceVR aparece en él o puede perder el acceso al grupo. Si ha vinculado su cuenta con La vinculación de cuentas Microsoft, deberá asegurarse de que la dirección de correo electrónico usada es la dirección de la cuenta Microsoft.  
    * Al pegar direcciones en su campo, asegúrese de que hay un correo electrónico por línea.  
    * Esta lista acepta correos electrónicos sin una cuenta altspaceVR válida.Se comprueban los nombres de usuario de AltspaceVR, las direcciones de correo electrónico no. Sin embargo, cuando se usa ese correo electrónico para crear una cuenta altspaceVR, el usuario al que está asociado obtendrá acceso al grupo.  
    * Puede importar una lista desde un archivo *.csv. Seleccione la opción "Elegir archivo" y seleccione el archivo.
    * Se recomienda usar correos electrónicos de miembro como forma de administrar grupos grandes, ya que es una manera más definitiva de conciliar a los miembros.Tenga en cuenta que los administradores de grupo tendrán que informar a sus usuarios o miembros de que deben usar la misma dirección de correo electrónico EXACTA que usaron en la plataforma de pago para crear una cuenta altspaceVR.  
    * Consulte la sección Importar usuarios en grupos para ver instrucciones sobre cómo obtener la lista de usuarios.   
* **Habilitación de la importación automática**   : la importación automática es un proceso periódico que se ejecuta en la lista de permitidos por correo electrónico. Concilia si un correo electrónico tiene ahora una cuenta de AltspaceVR válida asociada y concede acceso.   
    * Se recomienda habilitarla si usa una carga de archivos CSV.
* **Id. de vídeo de**   YouTube : permite insertar un vídeo de YouTube en la página de aterrizaje del grupo en el sitio web.Tenga en cuenta que esto es solo para el sitio web de grupo y no aparece dentro de AltspaceVR.
    * Debe usar el identificador de vídeo, no la dirección completa. El identificador de vídeo es el conjunto de caracteres que aparecen después del inicio de sesión "=" (igual) en YouTube.  
* **Identificador de Twitter**   : inserte los tweets más recientes de una cuenta de Twitter en la página de aterrizaje del grupo en el sitio web.Tenga en cuenta que esto es solo para el sitio web de grupo y no aparece dentro de AltspaceVR.
* **Crear grupo**   : botón situado en la parte inferior de la página. Cuando haya completado los campos descritos anteriormente, seleccione este botón para crear el grupo.  

### <a name="im-having-trouble-managing-my-groups-what-do-i-do"></a>Tengo problemas para administrar mis grupos. ¿Qué puedo hacer?

Si es miembro o administrador de un grupo, tiene herramientas adicionales para administrar los eventos. Para acceder a estas opciones, vaya a la página Grupos del sitio [web](https://account.altvr.com/groups).  

1. Seleccione Miembro o Administrador en la barra superior y seleccione el grupo que desea administrar o actualizar. 
2. Una vez en la página Miembro o Administrador, seleccione el grupo al que desea acceder haciendo clic en su imagen de icono.

![Panel de miembro o administrador abierto](images/group-features-img-08.png)

Esta es la página principal para ver todas las pertenencias a grupos como miembro o administrador.Seleccione un grupo para ver sus opciones específicas.

![Pantalla de pertenencia a grupos](images/group-features-img-09.png)

La página Grupo que ve cuando es miembro de un grupo.

**Como miembro:**

* Puede acceder a la página de aterrizaje del grupo en el sitio web para ver las actualizaciones del grupo.* Tiene un botón para "Salir" del grupo.Una vez hecho esto, se mostrará lo siguiente:  
    * Ya no tendrá acceso a la página de aterrizaje del grupo.  
    * Ya no verá ni escribirá el grupo "Commons" world.  
    * No puede ver eventos de grupo.

![Página Eventos de grupo](images/group-features-img-10.png)

La página de grupo que ve cuando es administrador de un grupo.Tenga en cuenta que verá las opciones de administrador independientemente de cómo escriba la página del grupo.

**Como administrador:**   Tiene las mismas opciones que un miembro, pero tiene funcionalidades adicionales.  

* Puede modificar todas las opciones que configuró anteriormente para crear el grupo.Vaya al botón "Editar" de la página del grupo para acceder a ellos.  
    * **Agregar otros administradores para un grupo:**   Al editar, ahora tiene la opción de agregar otros usuarios como administradores para ayudar a administrar el grupo.  

> [!NOTE]

* También tiene acceso a un **botón "Crear** evento".

    * La documentación detallada para crear un evento a través del sitio web está [aquí.](https://help.altvr.com/hc/en-us/articles/360033140854-Creating-an-Event)

* Esto solo está disponible para el propietario del grupo, es decir, el usuario que creó originalmente el grupo.
* No está disponible para los administradores del grupo.
