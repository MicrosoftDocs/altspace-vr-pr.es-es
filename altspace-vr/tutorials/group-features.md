# <a name="using-the-groups-feature-to-create-private-events"></a>Usar la característica grupos para crear eventos privados

Nuestra característica de grupos proporciona a los hosts la posibilidad de crear un espacio de VR abierto a una lista aprobada de invitados. Los hosts pueden agregar manualmente cualquier nombre de usuario o correo electrónico de AltspaceVR que les gustaría conceder acceso a su espacio. Aquí se muestra cómo puede usar esta característica para hospedar eventos, reuniones y conferencias:

> [!NOTE]
> Antes de tener acceso a "grupos", debe participar en las características de acceso temprano iniciando sesión en altvr.com, editando el perfil de usuario en el sitio web y seleccionando acceso temprano. O dentro de la aplicación AltspaceVR, configuración, general, participar en el programa de acceso temprano.

1. En el explorador, vaya a  [https://altvr.com](https://altvr.com)   e inicie sesión en su cuenta. 
2. Busque el menú desplegable **más** .
3. Seleccionar **grupos**

![AltspaceVR vista Web con más menús desplegables resaltados](images/group-features-img-01.png)

4. Una vez en la página grupos, puede ver una lista de los grupos de los que forma parte o un administrador. 

![AltspaceVR página de grupos de vistas Web](images/group-features-img-02.png)

5. Seleccione en el botón **crear** para crear un grupo nuevo.
6. Rellene los campos para crear el grupo.

![Página creación de nuevo grupo con campos de entrada](images/group-features-img-03.png)

> [!NOTE]
> Puede mantener el mouse sobre el control verde "?" para obtener información adicional acerca de los campos que los tienen.

7. Envíe un correo electrónico a todos los asistentes y pídale que cree una cuenta de AltspaceVR con la misma dirección de correo electrónico. Pueden crear una cuenta de AltspaceVR en el mundo o en nuestro [sitio web](https://account.altvr.com/users/sign_up).

## <a name="how-to-create-a-private-eventfrom-a-group"></a>Cómo crear un evento privado desde un grupo 

1. Una vez en la página miembro o administrador, seleccione el grupo al que desea obtener acceso. 
2. **Como administrador**, puede modificar todas las opciones que configuró anteriormente para crear el grupo.Vaya al botón "Editar" en la página del grupo para acceder a ellos.  
3. **Agregar otros administradores para un grupo:**   Al editar, puede agregar otros usuarios como administradores para ayudar a administrar el grupo.  
    * Debe ser el nombre de usuario de AltspaceVR para el usuario y no su cuenta de correo electrónico.  
4. Seleccione **Crear evento**.  

![Evento creado](images/group-features-img-04.png)

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>Cómo usar Patreon y eventbrite para hospedar un evento de VR con vales: 

Puede usar Patreon, eventbrite para crear eventos con vales en Altspace.Tendrá que crear una lista de direcciones de correo electrónico de los usuarios o exportar esa información de la plataforma de pago.  

### <a name="creating-a-list"></a>Crear una lista

Use un programa de hoja de cálculo para crear una sola columna con todas las direcciones de correo electrónico. Una vez hecho esto, haga lo siguiente:  

1. Cree o edite el grupo.  
2. Busque el campo de **correos electrónicos de miembros** .  
3. Busque la opción **elegir archivo** para importar los mensajes de correo electrónico de un archivo *. csv.
4. La lista se debe importar con un correo electrónico por línea.

> [!NOTE]
> Si lo desea, también puede copiar y pegar correos electrónicos en el campo en lugar de importarlos desde un archivo *. csv.

### <a name="importing-a-list"></a>Importar una lista

Las plataformas como Patreon y eventbrite permiten exportar un archivo *. csv (valor separado por comas) que tiene una columna de "correo electrónico" dedicada.Puede seleccionar esta columna para su uso.Una vez que esté disponible, los pasos son principalmente los mismos:  

1. Cree o edite el grupo.  
2. Busque el campo de **correos electrónicos de miembros** .  
3. Busque la opción **elegir archivo** para importar los mensajes de correo electrónico de un archivo *. csv.
4. La lista se debe importar con un correo electrónico por línea.

> [!NOTE]
> Si lo desea, también puede copiar y pegar correos electrónicos en el campo en lugar de importarlos desde un archivo *. csv.

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>Quiero usar eventbrite y Patreon, pero no veo dónde obtener las direcciones de correo electrónico para mis grupos

**Patreon ** 

[Patreon](https://www.patreon.com/)   es una plataforma de pertenencia en la que los creativos pueden crear un programa de suscripción para su contenido.Aunque Patreon tiene varias maneras de exportar datos de donativos, nos centraremos en las direcciones de CSV. Puede encontrar su página de ayuda para esta información [aquí](https://www.patreon.com/portal/how-to/export-pledge-data)   o puede seguir las instrucciones siguientes:  

1. Busque el campo "patrocinadores" en la barra lateral del creador a la izquierda del sitio de Patreon.  
2. Expándalo para buscar el "Administrador de relaciones" y selecciónelo.  
3. En la parte superior de la página, es un botón Descargar denominado CSV. Selecciónela.  
4. Aparecerá un campo de cuadro de diálogo de Windows para guardar el archivo.Guarde y  
5. Siga las instrucciones de importación de la página del grupo AltspaceVR para usar este archivo.  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)   es un sitio web de vales que permite a los usuarios administrar y crear vales para eventos.Al igual que Patreon, hay muchas maneras de generar listas de correo electrónico de asistentes.Nos centraremos en las direcciones de CSV. Puede encontrar su página de ayuda para esta información [aquí](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)   o puede seguir las instrucciones a continuación para obtener una manera de extraer estos datos.  

1. En el menú de perfil, busque el menú "administrar eventos". 
2. Seleccione el evento en el que desea obtener la lista de correo electrónico del asistente. 
3. En el panel del evento, busque la categoría denominada "informes comunes". 
4. También puede encontrar "informes comunes" en el menú de la columna izquierda, en "analizar-> informes de eventos".   
5. Seleccione "Resumen de asistentes". 
6. Busque la sección "resultados del informe". 
7. Seleccione el botón "CSV" para descargar el archivo.
