# <a name="using-the-groups-feature-to-create-private-events"></a>Uso de la característica Grupos para crear eventos privados

Nuestra característica grupos ofrece a los hosts la capacidad de crear un espacio de realidad virtual que esté abierto a una lista aprobada de invitados. Los hosts pueden agregar manualmente los nombres de usuario o correos electrónicos de AltspaceVR a los que les gustaría conceder acceso a su espacio. Aquí se muestra cómo puede usar esta característica para hospedar eventos, reuniones y conferencias:

> [!NOTE]
> Para poder acceder a "Grupos", debe participar en las características de acceso anticipado iniciando sesión en altvr.com, editando el perfil de usuario en el sitio web y seleccionando Acceso anticipado. O bien, dentro de la aplicación AltspaceVR, Configuración, General, Participar en el programa de acceso anticipado.

1. En el explorador, vaya a  [https://altvr.com](https://altvr.com)   e inicie sesión en su cuenta. 
2. Busque el **menú** desplegable Más.
3. Seleccionar **grupos**

![AltspaceVR web view with more pull down menu highlighted (AltspaceVR web view with more pull down menu highlighted )AltspaceVR web view with more pull down menu highlighted](images/group-features-img-01.png)

4. Una vez en la página Grupos, puede ver una lista de grupos de los que forma parte o de los que es administrador. 

![AltspaceVR web view groups page (Página de grupos de vistas web de AltspaceVR)](images/group-features-img-02.png)

5. Seleccione el botón **Crear** para crear un nuevo grupo.
6. Rellene los campos para crear el grupo.

![Nueva página de creación de grupos con campos de entrada](images/group-features-img-03.png)

> [!NOTE]
> Puede mantener el puntero sobre el verde "?" para obtener información adicional sobre los campos que los tienen.

7. Envíe un correo electrónico a todos los asistentes y haga que creen una cuenta altspaceVR con la misma dirección de correo electrónico. Pueden crear una cuenta altspaceVR en el mundo o en nuestro sitio [web.](https://account.altvr.com/users/sign_up)

## <a name="how-to-create-a-private-eventfrom-a-group"></a>Cómo crear un evento privado a partir de un grupo 

1. Una vez en la página Miembro o Administrador, seleccione el grupo al que desea acceder. 
2. **Como administrador,** puede modificar todas las opciones que configuró anteriormente para crear el grupo.Vaya al botón "Editar" de la página del grupo para acceder a ellos.  
3. **Agregar otros administradores para un grupo:**   Al editar, puede agregar otros usuarios como administradores para ayudar a administrar el grupo.  
    * Debe ser el nombre de usuario altspaceVR del usuario y no el correo electrónico de su cuenta.  
4. Seleccione **Crear evento**.  

![Evento creado](images/group-features-img-04.png)

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>Cómo usar Patreon y Eventbrite para hospedar un evento de VR con vales: 

Puede usar Patreon, Eventbrite para crear eventos con vales en Altspace.Deberá crear una lista de las direcciones de correo electrónico de los usuarios o exportar esa información desde la plataforma de pago.  

### <a name="creating-a-list"></a>Creación de una lista

Use un programa de hoja de cálculo para crear una sola columna con todas las direcciones de correo electrónico. Una vez que lo haya hecho, haga lo siguiente:  

1. Cree o edite el grupo.  
2. Busque el campo **Mensajes de correo electrónico de** miembro.  
3. Busque la **opción Elegir archivo** para importar los correos electrónicos desde un archivo *.csv.
4. La lista debe importarse con un correo electrónico por línea.

> [!NOTE]
> Si lo elige, también puede copiar y pegar correos electrónicos en el campo en lugar de importar desde un archivo *.csv.

### <a name="importing-a-list"></a>Importación de una lista

Plataformas como Patreon y Eventbrite permiten exportar un archivo *.csv (valor separado por comas) que tiene una columna dedicada de "correo electrónico".Puede seleccionar esta columna para su uso.Una vez que esté disponible, los pasos son principalmente los mismos:  

1. Cree o edite el grupo.  
2. Busque el campo **Mensajes de correo electrónico de** miembro.  
3. Busque la **opción Elegir archivo** para importar los correos electrónicos desde un archivo *.csv.
4. La lista debe importarse con un correo electrónico por línea.

> [!NOTE]
> Si lo elige, también puede copiar y pegar correos electrónicos en el campo en lugar de importar desde un archivo *.csv.

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>Quiero usar Eventbrite y Patreon, pero no veo dónde obtener las direcciones de correo electrónico de mis grupos.

**Patreon ** 

[Patreon](https://www.patreon.com/)   es una plataforma de pertenencia en la que los creativos pueden crear un programa de suscripción para su contenido.Aunque Patreon tiene varias maneras de exportar datos complicados, nos centraremos en las direcciones de CSV. Puede encontrar su página de ayuda para esta información [aquí](https://www.patreon.com/portal/how-to/export-pledge-data)   o puede seguir las instrucciones siguientes:  

1. Busque el campo "Lores" en la barra lateral del creador, a la izquierda del sitio de Patreon.  
2. Expándalo para buscar el "Administrador de relaciones" y selecciónelo.  
3. En la parte superior de la página, hay un botón de descarga denominado CSV. Selecciónela.  
4. Se abrirá un campo de cuadro de diálogo de Windows para guardar el archivo.Guarde y  
5. Siga las instrucciones de importación de la página del grupo AltspaceVR para usar este archivo.  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)   es un sitio web de vales que permite a los usuarios administrar y crear incidencias para eventos.Al igual que Patreon, hay muchas maneras de generar listas de correo electrónico de asistentes.Nos centraremos de nuevo en las direcciones de CSV. Puede encontrar su página de ayuda para esta información [aquí](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)o puede seguir las instrucciones siguientes para obtener una   manera de extraer estos datos.  

1. En el menú Perfil, busque el menú "Administrar eventos". 
2. Seleccione el evento en el que desea obtener la lista de correo electrónico del asistente. 
3. En el panel del evento, busque la categoría denominada "Informes comunes". 
4. También puede encontrar "Informes comunes" en el menú de columnas de la izquierda en "Analizar -> informes de eventos".   
5. Seleccione "Resumen del asistente". 
6. Busque la sección "Resultados del informe". 
7. Seleccione el botón "CSV" para descargar el archivo. 

> [!IMPORTANT]
> Se trata de una característica beta. AltspaceVR es una experiencia gratuita y no es responsable de los problemas técnicos que puedan surgir con la experiencia que podrían impedir que los asistentes accedan a eventos con vales.
